1. DESCRIPTION

    Project name : LIKEO

    Brief overview and goal :
        LIKEO is a social media platform that allows people to connect, communicate and share content online

    General features :
        - User profile : User can share posts and photos on his personnal page
        - News feed : User can see the other users's activities like post on his own page
        - Friends and connections : User can send friend requests, accept or remove friends
        - Like : User can live a like to his friend's posts
        - Chat : Users can send message to each other
